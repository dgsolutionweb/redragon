# Story 001 - Gamer Bio Card Vermelho/Preto

## Status
Done

## User Story
Como criador de conteudo, eu quero uma pagina unica para link da bio com visual gamer vermelho e preto, para direcionar visitantes para meus principais links.

## Acceptance Criteria
1. A pagina deve usar identidade visual dominante em vermelho e preto.
2. A tipografia deve ter estilo gamer.
3. Deve existir uma sequencia inicial de "boot" de computador.
4. Deve haver efeitos de luz vermelha animados no fundo.
5. O card principal deve exibir links clicaveis para redes/plataformas.
6. O layout deve funcionar em desktop e mobile.
